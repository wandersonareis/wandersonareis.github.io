API Dindin 2.0 / Desafio de aprendizado
API Dindin 2.0
Mar 2023 - Mar 2023 - Atual

Durante o desafio Back-end do Módulo 3, meu objetivo era criar uma API RESTful capaz de realizar diversas operações relacionadas a usuários e transações. A API desenvolvida permite o cadastro e login de usuários, bem como a edição de informações pessoais. Também é possível listar transações, detalhar transações específicas e cadastrar novas transações. Além disso, a API oferece a funcionalidade de edição, remoção e obtenção de extrato de transações, incluindo a possibilidade de filtrar transações por categoria. No geral, foi um projeto desafiador e extremamente gratificante. Espero ter conseguido atender aos requisitos do desafio e, acima de tudo, proporcionar uma experiência de usuário fluída e completa.
